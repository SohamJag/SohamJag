## 🏆 Featured Projects & Portfolio

### [🌐 NetSage: AI NOC Troubleshooting Assistant](https://github.com/SohamJag/NetSage-AI-Powered-Network-Troubleshooting-Assistant)
An enterprise-grade, full-stack network assurance platform that automates virtual router diagnostics.
*   **Core stack:** React 19, FastAPI, SQLite, Netmiko, Ollama (Llama 3), CSS3 Grids.
*   **Key Feature:** Implements **50 predefined logic-based network failure rules** matching OSPF area type mismatches, administrative shutdowns, and CPU exhaustions, and enriches them dynamically via a local LLM technical advisor.
*   **Visual Interface:** Interactive SVG topology map with animated packet flows and retro Cisco CLI terminal emulation proxy.

### [🔒 Obsidian Circuit: Static CTF Security Challenge](https://github.com/SohamJag/obsidian-circuit)
A static, Netlify-ready CTF challenge portal testing candidate engineering skills.
*   **Core stack:** HTML5, Vanilla JavaScript, TailwindCSS, Webpack, Obfuscator-CLI.
*   **Key Feature:** Implements access control, information leakage, cookie manipulation, and log analysis challenges with high-intensity code obfuscation to prevent source inspection.

### [📈 AI-Secure Expense Manager](https://github.com/SohamJag/ai-secure-expense-manager)
A secure financial dashboard powered by an anomaly detection microservice.
*   **Core stack:** Python (Isolation Forest / PyTorch), Node.js, Express, MongoDB, React, JWT, Rate Limiting.
*   **Key Feature:** Automatically flags anomalous, potentially fraudulent transaction drift while enforcing modern JWT cookie sessions and security headers (Helmet).

### [🚀 CI/CD Docker AWS App](https://github.com/SohamJag/ci-cd-docker-aws-app)
A multi-stage automated deployment pipeline.
*   **Core stack:** Docker, GitHub Actions, AWS EC2, PM2, Node.js, Express.
*   **Key Feature:** Automatically tests, builds container images, pushes to registries, and deploys asynchronously to AWS cloud instances upon master merges.

### [🏹 Sentinel RDP Snare](https://github.com/SohamJag/Sentinel-RDP-Snare)
A physical Azure RDP honeypot setup logging global brute force attempts in real-time.
*   **Core stack:** Microsoft Sentinel (SIEM), Log Analytics Workspaces, KQL (Kusto Query Language), PowerShell geolocation API.
*   **Key Feature:** Captures Windows Event logs from exposed VMs, resolves attacker IP locations via external APIs, and plots geographic attack densities in a custom Sentinel dashboard.
